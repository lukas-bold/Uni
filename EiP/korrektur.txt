Aufgabe 1: Algorithmen [Pkte 5/6]
a) [0.5/1]
-> 1*1 = 1, sprich 1 ist eine Quadratzahl und die Methode sollte
hier true zurückgeben. Deine Implementierung liefert hier false (auch bei Eingabe 4 etc.).
Können wir aber ganz einfach reparieren, indem wir in der
while-Bedingung ein '>=' statt '>' verwenden.

b) [1.5/2]
Beachte, dass das erste Element der fibonacci Reihe 0 ist (s.Angabe)
Mit:
if (i==0 || i==1) {
 arr[i] = 1;
}
setzt du auch das erste Element auf 1.
Auch das lässt sich einfach durch einen zusätzlichen if-fall abfangen:
if (i==0) arr[i] = 0;
else if( i==1) arr[i] = 1;
...
